CSS - Span

  • Voltar

O elemento SPAN é usado para separar logicamente o conteúdo em linha.

Por exemplo, suponha que você queira enfatizar as qualidades de uma pessoa.

No parágrafo em que descreve as caraterísticas da pessoa, use span para enfatizar as qualidades de um objeto/pessoa.

O CSS é fácil e simples de usar.
