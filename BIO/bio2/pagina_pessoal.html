PopcornNow

@Instituto Federal de Educação, Ciência e Tecnologia de São Paulo - IFSP

Aplicação web desenvolvida em equipe para a disciplina de Programação Orientada a Objetos. PopcornNow é uma aplicação web que permite que os usuários tenham um controle e organização sobre as séries televisivas que estão assistindo, já assistiram ou pretendem assistir. Utilizou-se a linguagem de programação Java e o framework Spring Boot para o desenvolvimento do back-end em API REST, HTML, TypeScript e Angular no front-end e integração com a API de filmes e séries TMDb.

Strigo - Ambiente Virtual de Aprendizagem baseado no Ensino Colaborativo

@Colégio Técnico de Campinas - Unicamp

Strigo é uma plataforma educacional baseada no ensino colaborativo que visa quebrar barreiras educacionais existentes atualmente, possibilitando que os colaboradores compartilhem conhecimentos culturais, técnicos e científicos. Para tornar isso possível, foi desenvolvida uma Application Programming Interface (API) com o framework Laravel 5, utilizando a arquitetura Model View Controller (MVC) e a ferramenta AltivaJS.
